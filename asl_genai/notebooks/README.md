# Generative AI Notebooks

This directory contains a collection of Jupyter notebooks designed to guide you through the fundamentals and advanced topics of Generative AI on Google Cloud.

To get the most out of these materials, we recommend following the learning path in the order listed below.

## Recommended Learning Path

### 1. [Generative AI Fundamentals](./generative_ai_fundamentals/)
Learn the core concepts of Generative AI, including prompt engineering, multimodal capabilities, and how to use and tune Gemini on Google Cloud.

*   **Focus Areas:**
    *   Prompt Engineering
    *   Multimodal Prompting
    *   LangChain Integration
    *   Model Tuning on Google Cloud

### 2. [Retrieval Augmented Generation](./retrieval_augmented_generation/)
Understand how to ground your language models with external data to reduce hallucination and provide up-to-date information.

*   **Focus Areas:**
    *   Semantic Matching
    *   Core RAG concepts
    *   Vector Search on Google Cloud

### 3. [Building Agents](./building_agents/)
Explore how to build autonomous agents that can use tools and perform complex tasks.

*   **Focus Areas:**
    *   Function Calling
    *   Building Agents with ADK (Agent Development Kit)
    *   Designing Complex Multi-Agent Workflows

### 4. [Operationalize Gen AI](./operationalize_gen_ai/)
Learn how to evaluate and operationalize your Generative AI applications for production.

*   **Focus Areas:**
    *   Gen AI Evaluation Services
    *   Deploying Agents on Google Cloud
